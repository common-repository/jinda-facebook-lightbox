== Description ==

Jinda Facebook lightbox คือปลั๊กอินเอาไว้แสดงผล Facebook Page Plugin ใน lightbox(หรือที่เรียกกันว่า popup) ได้อย่างสวยงามใน WordPress ด้วยปลั๊กอิน Jinda Facebook lightbox นี้คุณสามารถตั้งค่าพื้นฐานในเรื่องของการแสดงผลได้ด้วยตัวเอง ไม่ว่าจะเป็นสีของปุ่ม, ตัวหนังสือ รวมไปถึงการตั้งค่าเนื้อหาที่จะแสดงก่อน และหลังของ Page Plugin ได้อย่างอิสระ ปลั๊กอินนี้เป็น lightbox ที่สร้างมาเพื่อรองรับขนาดหน้าจอทุกประเภท หรือที่เรียกกันว่า responsive layout

Jinda Facebook lightbox is built for website that want to show Facebook Page Plugin in simple backdrop lightbox. with responsive ready and free to custom some content and color. A smart simple way to show Facebook Page Plugin in lightbox to your website. with Jinda Lightbox, You can custom some content that show before and after Page Plugin or invetal and delay for lightbox.

ความสามารถ(Features):

*   Fully Responsive WordPress Lightbox
*   Easy to install, use and settings
*   You can override custom CSS
*   Set content above and below the Facebook Page Plugin
*   You can select page that want popup to show (Homepage, Post, Page)
*   Set timer that you wanna wait popup to show
*   Set expire time that you wanna show popup to your visitor

This plugin was created by [JindaTheme](http://www.jindatheme.com). check our Github repo to see more our opensource libraries.

== Installation ==

First download the Jinda Facebook lightbox plugin ZIP file from this page.

1. Upload `jinda-facebook-lightbox` to the `/wp-content/plugins/` directory
1. Login to your WordPress dashboard with admin privileges.
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to settings tab > Jinda Lightbox and set value that you prefer.

== Frequently Asked Questions ==

= Jinda Facebook Lightbox ต่างจาก Facebook Lightbox อื่นยังไง? =

เราสร้าง Jinda Facebook Lightbox ขึ้นมาให้มีหน้าตาที่สวยงาม สามารถแสดงผลได้กับทุกอุปกรณ์ และยังสามารถตั้งค่าการใช้งานได้อย่างอิสระ ทั้งสามารถใส่เนื้อหาลงไปใน lightbox เองได้

= Jinda Facebook Lightbox ใช้กับ web browser ไหนได้บ้าง? =

Web browser ที่สามารถใช้งาน Jinda Facebook Lightbox ได้อย่างเต็มประสิทธิภาพได้แก่ Google Chrome, Firefox, Safari และ Internet Explorer เวอร์ชั่น 9 ขึ้นไป

= Jinda Facebook Lightbox รองรับอุปกรณ์อื่นนอกเหนือจากคอมพิวเตอร์? =

ใช่! Jinda Facebook Lightbox เป็นปลั๊กอินที่สามารถเปิดใช้งานได้กับทุกขนาดหน้าจอของอุปกรณ์ที่เปิดไม่ว่าจะเป็น Mobile, Tablet ไปจนถึง Large Desktop เรียกได้ว่าเป็น responsive WordPress lightbox

= หากฉันต้องการปรับแต่ง Jinda Facebook Lightbox ล่ะ? =

คุณสามารถใส่ค่า CSS ที่ต้องการลงไปในเมนูการตั้งค่าของ Jinda Facebook Lightbox ในหัวข้อ Your custom CSS ได้เลย

= ฉันมีข้อสงสัย หรือพบเจอปัญหาในการใช้งาน =

สามารถแจ้งปัญหาการใช้งานของ Jinda Facebook Lightbox เข้ามาได้ที่เว็บไซต์หลักของ [JindaTheme](http://www.jindatheme.com) ได้ทันที
